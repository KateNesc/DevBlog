## 📝 Завдання до роботи
Розробити статичні Web-сторінки додатку та перевірити коректне їх відображення на екранах різного розміру (ПК, планшети, смартфони).

**Обрана тематика:** Блог з коментарями (DevBlog).
Додаток дозволяє переглядати стрічку новин, додавати нові повідомлення та коментарі до них, а також реалізує можливість їх вилучення.

## 🚀 Посилання на ресурси
* **GitHub Pages (Сайт):** [[ПОСИЛАННЯ_НА_САЙТ]](https://katenesc.github.io/DevBlog/)

## 🛠 Технологічний стек
* **Розмітка:** HTML5
* **Стилізація:** CSS3
* **Фреймворк:** Tailwind CSS (для побудови адаптивного інтерфейсу)
* **IDE:** VS Code

## 📂 Структура проекту
Веб-сайт складається з наступних обов'язкових сторінок:
* `index.html` — Робоча сторінка (Стрічка блогу)
* `profile.html` — Профіль користувача (табличний вигляд даних)
* `about.html` — Про додаток (логотип та опис)
* `login.html` — Вхід до сайту
* `registration.html` — Реєстрація нового користувача
